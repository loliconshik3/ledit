<h1 align="center">Standart key bindings</h1>

# Work with files

* `Ctrl+n` - New file.
* `Ctrl+o` - Open file.
* `Ctrl+s` - Save file.
* `Ctrl+Shift+s` - Save as.
* `Ctrl+k` - Open directory.

# Work with text

* `Ctrl+a` - Select all text.
* `Ctrl+v` - Insert copied text.
* `Ctrl+=` - Up text zoom.
* `Ctrl+-` - Down text zoom.

# Work with editor

* `Ctrl+e` - Set focus on command line.
